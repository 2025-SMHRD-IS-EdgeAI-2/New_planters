# GreenSync (팀명: 식집사들)
<img width="1031" height="571" alt="image" src="https://github.com/user-attachments/assets/4175e55c-68e3-4e4f-aefb-6ae7da7cf8f3" />


## 👀 서비스 소개
* 서비스명:  스마트 식물집사 서비스
* 서비스설명: RasberryPi4 기반으로 센서 및 이미지 데이터 수집하여, OpenCV/OpenAI를 활용한 식물 성장치 분석, 이벤트 알림 및 식집사 숙련도 점수 서비스 제공
<br>

## 📅 프로젝트 기간
2026.01.26 ~ 2026.02.13 (2주)
<br>

## ⭐ 주요 기능
* 기능1 : 실시간 환경 데이터 제공
* 기능2 : 데이터 통계 - 시각화
* 기능3 : 이미지 타임랩스 - 영상화
* 기능4 : 성장 다이어리 (타임라인)
* 기능5 : 식물 성장 분석 (OpenCV)
* 기능6 : 식집사 숙련도 점수 (OpenAI)
<br>
